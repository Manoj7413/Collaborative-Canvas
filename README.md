# Collaborative Drawing Canvas

A real-time multi-user drawing application built with TypeScript, HTML5 Canvas, and WebSockets.

## Features

- **Real-time Drawing**: Multiple users can draw simultaneously on the same canvas
- **Drawing Tools**: Brush and eraser with adjustable size and color
- **User Management**: See who's online with colored indicators
- **Global Undo/Redo**: Undo and redo actions across all users
- **Cursor Tracking**: See where other users are drawing
- **Room Support**: Isolated drawing sessions
- **Mobile Support**: Touch-enabled drawing
